Características técnicas

Esta guía fue desarrollada en HTML, lo cual indica que puede ser visualizada como una página web común.

Condiciones recomendadas

Para una mejor visualización de esta guía se recomiendan las siguientes condiciones:

  1. Navegador Firefox versión 3.5 o superior.
  2. Resolución de vídeo de 1024 X 768.
  3. Monitor a color.

Si tiene problemas en visualizar esta guía adecuadamente o desconoce la manera de constatar si su equipo cumple con las condiciones recomendadas, consulte la ayuda de un experto.
